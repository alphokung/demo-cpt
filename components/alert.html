Alert & Confirm

ตัวอย่างการใช้งาน ThaiAlert — dialog แจ้งเตือนและยืนยันการดำเนินการ

Alert

แสดงข้อความแจ้งเตือนฝ่ายเดียว — ผู้ใช้กดปุ่มเพื่อปิด

Type Variants

Confirm

ขอการยืนยันจากผู้ใช้ก่อนดำเนินการ — คืนค่า true เมื่อยืนยัน, false เมื่อยกเลิก

Type Variants

วิธีใช้งาน

1 · เพิ่ม script ในหน้าที่ต้องการใช้

<script src="../alert.js"></script>

2 · Alert — แสดงข้อความแจ้งเตือน

await ThaiAlert.alert({
  type: 'success',       // 'info' | 'success' | 'warning' | 'danger'
  title: 'บันทึกสำเร็จ',
  message: 'ข้อมูลถูกบันทึกเรียบร้อยแล้ว',
  buttonText: 'รับทราบ' // default: 'ตกลง'
});

3 · Confirm — ขอการยืนยันก่อนดำเนินการ

const confirmed = await ThaiAlert.confirm({
  type: 'danger',
  title: 'ลบรายการนี้?',
  message: 'การดำเนินการนี้ไม่สามารถกู้คืนได้',
  confirmText: 'ลบ',     // default: 'ยืนยัน'
  cancelText: 'ยกเลิก'  // default: 'ยกเลิก'
});
if (confirmed) { /* ผู้ใช้กดยืนยัน */ }
