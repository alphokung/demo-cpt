การใช้งานพื้นฐาน

<span class="material-symbols-outlined">home</span>

โหลด font ครั้งเดียวใน <head> ของทุกหน้า (มีอยู่แล้วใน styles.scss)

<!-- ไม่ต้องเพิ่มเองถ้าใช้ styles.css — ถูก import ไว้แล้ว -->
@import url('https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200');

Variable Font Axes

Material Symbols เป็น variable font ปรับได้ 4 axes ผ่าน font-variation-settings

FILL — 0 (outlined) / 1 (filled)

favorite FILL 0
Outlined
favorite FILL 1
Filled
star FILL 0
star FILL 1
notifications FILL 0
notifications FILL 1

wght — น้ำหนักเส้น 100 (บาง) → 700 (หนา)

home 100
home 300
home 400
(default)
home 600
home 700

opsz — optical size ปรับ detail ตามขนาดที่ใช้จริง (20–48)

info opsz 20
(small)
info opsz 24
(default)
info opsz 40
info opsz 48
(large)

Live Customizer

ปรับค่า axes แบบ real-time และดู CSS ที่ได้

bookmark bookmark

CSS output

.material-symbols-outlined {
  font-size: 64px;
  font-variation-settings: 'FILL' 0, 'wght' 400, 'GRAD' 0, 'opsz' 24;
}

วิธีใช้งาน

พื้นฐาน — outlined (ค่าเริ่มต้น)

<span class="material-symbols-outlined">icon_name</span>

ปรับ axes ด้วย font-variation-settings

<!-- Filled, bold -->
<span class="material-symbols-outlined"
  style="font-variation-settings:'FILL' 1,'wght' 600,'GRAD' 0,'opsz' 24">
  favorite
</span>

<!-- หรือผ่าน CSS class -->
.icon-filled {
  font-variation-settings: 'FILL' 1, 'wght' 400, 'GRAD' 0, 'opsz' 24;
}

ใช้กับ token สีและขนาด

<span class="material-symbols-outlined"
  style="font-size:20px; color:var(--foreground-primary-default)">
  check_circle
</span>
