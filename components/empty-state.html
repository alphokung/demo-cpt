Empty State

แสดงเมื่อไม่มีข้อมูล — ไอคอน, หัวข้อ, คำอธิบาย และ action

Basic

ไอคอนวงกลมเทา + หัวข้อ + คำอธิบาย

inbox

ไม่พบรายการ

ยังไม่มีข้อมูลในระบบ

Context Variants

ปรับไอคอนและข้อความตามบริบทการใช้งาน

receipt_long

ไม่มีใบสั่งค้างชำระ

คุณไม่มีใบสั่งจราจรที่ยังไม่ได้ชำระ
ขับขี่อย่างปลอดภัยต่อไปนะ

folder_open

ยังไม่มีเอกสาร

เพิ่มเอกสารของคุณเพื่อเข้าถึงได้ทุกที่ทุกเวลา

search_off

ไม่พบผลการค้นหา

ไม่พบข้อมูลที่ตรงกับ "ใบขับขี่ประเภท ค."
ลองค้นหาด้วยคำอื่น

With Action Button

เพิ่มปุ่ม action ใต้คำอธิบาย

receipt_long

ยังไม่มีประวัติการชำระ

เมื่อคุณชำระค่าปรับแล้ว ประวัติจะแสดงที่นี่

wifi_off

ไม่สามารถโหลดข้อมูลได้

กรุณาตรวจสอบการเชื่อมต่ออินเทอร์เน็ต แล้วลองใหม่อีกครั้ง
