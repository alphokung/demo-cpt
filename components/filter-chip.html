Filter Chip

States — Default & Active

default: เส้นขอบเทา — active: พื้นหลังสีเขียวอ่อน

With Colour Dot

.filter-dot จุดสีระบุหมวดหมู่ก่อนข้อความ

Scrollable Row — filter-row

.filter-row ซ่อน scrollbar และ overflow-x auto

← scroll แนวนอนบนหน้าจอแคบ

Toggle active — JavaScript

function setActive(btn) {
  const row = btn.closest('.filter-row');
  row.querySelectorAll('.filter-chip')
    .forEach(c => c.classList.remove('active'));
  btn.classList.add('active');
}
