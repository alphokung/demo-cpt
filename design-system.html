คู่มือการใช้งานระบบดีไซน์ (Design System & CSS Guide)

รายละเอียดโทเค็นระบบดีไซน์, โค้ดสี, ฟอนต์การจัดองค์ประกอบ และการใช้งานไฟล์สไตล์ร่วมกันภายในโครงการ

การจัดการและคอมไพล์สไตล์ด้วย SCSS

โครงการนี้ใช้งาน Sass (SCSS) ในการจัดการไฟล์สไตล์เพื่อให้อ่านง่ายและบำรุงรักษาง่ายขึ้นผ่านโครงสร้างการเขียนสไตล์แบบซ้อนกัน (Nesting) และตัวแปรระบบ

# 1. ติดตั้งไลบรารีสไตล์ผ่าน npm npm install # 2. คอมไพล์ไฟล์ styles.scss ไปยัง styles.css (ครั้งเดียว) npm run sass:build # 3. ให้ระบบตรวจจับการแก้ไขไฟล์และคอมไพล์อัตโนมัติขณะพัฒนา (Live Watcher) npm run sass:watch

* ทุกหน้าเว็บหลักยังคงเรียกใช้งานไฟล์ styles.css เสมอ ดังนั้น ห้ามแก้ไขโค้ดที่ไฟล์ CSS โดยตรง ให้แก้ไขที่ไฟล์ styles.scss แทนเพื่อให้ตัวคอมไพเลอร์ทำงานและอัปเดตสไตล์อย่างถูกต้อง

ขนาดและรูปแบบอักษร (Typography)

ระบบดีไซน์เลือกใช้ฟอนต์ Anuphan (Google Fonts) โดดเด่นด้วยลายเส้นที่โค้งมน สะอาดตา และทันสมัย เหมาะกับระบบราชการยุคใหม่

Title Medium (16px - SemiBold)

ประวัติการหักและคืนคะแนน

Body Medium (14px - Regular)

ข้อมูลใบสั่งจราจร โดยสำนักงานตำรวจแห่งชาติ (ตร.) หากคะแนนของคุณถูกตัดจนเหลือศูนย์คะแนน จะถูกสั่งพักใช้ใบอนุญาตขับขี่
